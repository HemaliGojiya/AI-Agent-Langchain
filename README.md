# AI-Agent-Langchain
A full-stack GenAI chatbot application using LangGraph ReAct agents, FastAPI, and Streamlit. Supports Groq (LLaMA 3.3, Mixtral) and OpenAI (GPT-4o-mini) models with dynamic model selection, custom system prompts, and Tavily-powered web search. Designed with a clean 3-phase architecture: AI Agent → FastAPI Backend → Streamlit Frontend.
